text-decoration : 텍스트에 줄을 긋거나 없앨 때 사용

text-decoration: none|underline|overline|line-through;
  • 텍스트 영역 아래 줄 긋기

  • 텍스트 영역 중간에 줄 긋기

  • 텍스트 영역 위에 줄 긋기

text-transform : 영문 텍스트의 대소문자 변환시 사용

선택자 { text-transform: uppercase|lowercase|capitalize; }
  1. uppercase : 모든 영문자를 대문자로
  2. LOWERCASE : 모든 영문자를 소문자로
  3. capitalize : 영문자의 first text만 대문자로
