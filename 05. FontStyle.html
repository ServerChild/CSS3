font-size : 글꼴의 크기를 변경할 때 사용 

 선택자 { font-size:크기(px|em|%) }
  • 고정크기 px테스트
  • 가변크기 em테스트
  • 가변크기 %테스트

font-weight : 글꼴의 두께 지정 

 선택자 { font-weight:normal|bold|bolder|lighter|100|400|900; }
  • 굵은 글꼴로 변경
  • 원래 굵기보다 더 굵게
  • 원래 굵기보다 더 가늘게

font-variant : 영어 소문자를 작은 대문자로 변경할 때 사용 

 선택자 { font-variant:small-caps; }
